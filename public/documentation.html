Documentación de API

POST /api/auth/login

Endpoint para iniciar sesión y obtener un token de acceso.

Recibe en el body: { "username": "string", "passwd": "string" }

Devuelve en JSON:

Nombre Descripción
user_id uuid del usuario
token JWT que representa la sesion

Establece un JWT en la cookie 'access-token'.

Errores:

400 - Request malhecho o malformado

401 - Nombre o usuario incorrecto

POST /api/auth/logout

Endpoint para cerrar sesión.

Borra la cookie de sesion.

POST /api/auth/register

Endpoint para registrar un nuevo usuario.

Recibe en el body: { "username": "string", "passwd": "string" }

No devuelve ningún valor.

Errores:

400 - Request malhecho o malformado

401 - El usuario ya existe

GET /api/user

Endpoint para obtener datos del usuario.

Devuelve en JSON un objeto con los datos del usuario:

Nombre Descripción
section_id Identificador de la sección del usuario
cards Array de objetos de cartas del usuario
POST /api/user/section

Endpoint para crear una nueva sección para el usuario.

Recibe en el body: { "title": "string" }

Devuelve en JSON un objeto con los detalles de la sección creada.

Errores:

400 - Mal request, falta el titulo

DELETE /api/user/section

Endpoint para eliminar una sección del usuario.

Recibe en el body: { "section_id": "string" }

Devuelve en JSON un objeto con los detalles de la sección eliminada.

Errores:

401 - UUID incorrecta

500 - Seccion inexistente

403 - Seccion prohibida, modificacion no autorizada

POST /api/user/card

Endpoint para crear una nueva carta para el usuario.

Recibe en el body: { "title": "string", "content": "string", "section_id": "string" }

Devuelve en JSON un objeto con los detalles de la carta creada.

Errores:

400 - Mal request, faltan campos

401 - UUID incorrecta

403 - Seccion prohibida, modificacion no autorizada

DELETE /api/user/card

Endpoint para eliminar una carta del usuario.

Recibe en el body: { "card_id": "string" }

Devuelve en JSON un objeto con los detalles de la carta eliminada.

Errores:

400 - Reques malformado

401 - UUID incorrecta

500 - La carta no existe

403 - Seccion prohibida, modificacion no autorizada

PATCH /api/user/account

Endpoint para actualizar la cuenta de usuario.

Recibe en el body: { "username": "string", "email": "string", "password": "string" }

Devuelve en JSON un objeto con los datos de usuario actualizados.
